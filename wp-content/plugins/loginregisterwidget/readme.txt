Widget for login, register and lost password.

== Description ==
* This plugin adds a widget to your website that allows you to display a login form, register form and lost password form.
* You can disable Google ReCaptcha feature.
* The widget will not be shown if you are already logged in.
* No ads.
* No linkbacks.
* Google ReCaptcha requires PHP VERSION 5.3  or higher if not the recaptcha feature will not work.
* After you have installed the plugin a menu will appear Settings -> Login, Register Widget Settings
* Check the screenshots tab.
* If you finds bugs or need support go to the support tab.

== Installation ==

1. After you download the plugin go to Plugins -> Add New -> Upload Plugin
2. Choose the zip and upload
* After you have installed the plugin a menu will appear Settings -> Login, Register Widget Settings
* The widget will not be shown if you are already logged in.
